# Minecraft client chatlog

<p>A Python code for logging your Minecraft chat in real time in your CMD.</p>

<p>The code takes advantage of your chat log file, so <b>set the correct file path according to your Minecraft client.</b> It has a built in word blacklist, fill it in with your preferences ot delete the content of it. This is only a small Python code, it's not a professional solution.</p>
